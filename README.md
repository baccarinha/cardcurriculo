
# Currículo em Card

💕**Acesse aqui:**
[Currículo em Card](https://baccarinha.github.io/cardcurriculo/)

O projeto consiste em um card profissional digital, desenvolvido com o objetivo de apresentar informações pessoais e profissionais de forma rápida, visual e objetiva, funcionando como um cartão de apresentação online integrado ao currículo completo.

O sistema foi projetado para atuar como uma identidade profissional digital, reunindo dados essenciais como nome, área de atuação, objetivo profissional e formas de contato em um layout compacto e moderno. A proposta do projeto é facilitar o primeiro contato com recrutadores ou visitantes, permitindo que eles visualizem rapidamente o perfil profissional e tenham acesso direto ao currículo completo por meio de links integrados.

O card apresenta o nome Jéssica Thais Baccaro, destacando sua área de interesse em Tecnologia da Informação (TI) e sua atuação como Auxiliar de Auto Atendimento. Além disso, inclui um objetivo profissional, demonstrando interesse em crescer profissionalmente e contribuir com experiência em atendimento e afinidade com tecnologia. Esse tipo de informação ajuda a transmitir uma identidade profissional clara e direcionada ao mercado de trabalho.

Outro aspecto importante do projeto é a organização visual das informações, incluindo localização (Jundiaí), área de formação (ADS — Análise e Desenvolvimento de Sistemas) e canais de contato, como e-mail e link para currículo completo. Essa estrutura compacta segue o conceito de um cartão de apresentação digital, que permite acesso rápido às informações principais sem necessidade de leitura extensa.

Do ponto de vista técnico, o projeto demonstra a aplicação de desenvolvimento web front-end, com foco em layout responsivo e visual limpo. O sistema foi desenvolvido utilizando HTML para estruturação do conteúdo, CSS para estilização visual e JavaScript para possíveis interações e navegação, além de estar hospedado online por meio do GitHub Pages, permitindo acesso público e compartilhamento fácil por meio de link.

Outro diferencial do projeto é o uso de Inteligência Artificial (IA) como apoio na geração de ideias e estrutura do layout, demonstrando o uso de tecnologias modernas para criar soluções digitais práticas e funcionais. O card funciona como uma porta de entrada digital, conectando rapidamente o visitante ao currículo completo e reforçando a presença profissional online.

🧩 Resumo técnico do projeto

Nome do projeto:
Card Currículo Profissional

Objetivo:
Criar um cartão profissional digital para apresentar informações pessoais e facilitar o acesso ao currículo completo.

|Principais funcionalidades:|Tecnologias utilizadas:
|---|---|
Exibição de nome e área profissional|HTML
Apresentação de objetivo profissional|CSS
Exibição de localização|JavaScript
Indicação da formação acadêmica|GitHub Pages (hospedagem)
Link para currículo completo|Inteligência Artificial (apoio na criação do layout e ideias)
Acesso rápido ao contato por e-mail
Interface visual compacta e organizada
